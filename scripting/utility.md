##`I.wait`

Wait for a given amount of time

#### Usage
```javascript
I.wait(seconds)
```

##### Parameters

| Parameter | Type | Remarks|
|----------|------|--------|
| seconds | number | Number of seconds to wait for |

#### Example(s)

```javascript
I.wait(3);
```
Waits for 3 seconds.
